📄 Pagify

AI-Powered PDF Summarizer & Query Tool

"Education is the manifestation of the perfection already in man." — Swami Vivekananda

🚀 Live Demo

Try it here: pagify.aakarshtiwari.com

📦 Features

  • PDF Upload (up to 5MB, 10 pages)
  • AI Summarization using GPT-3.5-turbo
  • Ask Questions about the PDF
  • Text-to-Speech in Indian English (en-IN)
  • Responsive UI, Accessibility, SEO

🛠️ Tech Stack

  • Frontend: HTML, CSS, JS (Vanilla), Font Awesome
  • Backend: Node.js, Express.js
  • Database: MongoDB (Mongoose)
  • AI: OpenAI GPT-3.5-turbo
  • TTS: Web Speech API (en-IN)

📥 Installation

Frontend

git clone https://github.com/aakarshtiwari26/aipdfreader-frontend.git
cd aipdfreader-frontend

Backend

git clone https://github.com/aakarshtiwari26/aipdfreader-backend.git
cd aipdfreader-backend
npm install

.env Example

MONGODB_URI=mongodb+srv://:@cluster.mongodb.net/
OPENAI_API_KEY=sk-
FRONTEND_URL=https://pagify.aakarshtiwari.com

Run Server

npm run dev

🌐 Deployment

Vercel (Frontend)

{
  "version": 2,
  "builds": [
    { "src": "public/**", "use": "@vercel/static" }
  ],
  "routes": [
    {
      "src": "/api/(.*)",
      "dest": "https://aipdfreader-0f7m.onrender.com/api/$1"
    }
  ]
}

Render (Backend)

MONGODB_URI=mongodb+srv://:@cluster.mongodb.net/
OPENAI_API_KEY=sk-
FRONTEND_URL=https://pagify.aakarshtiwari.com

🧠 Usage

  • Upload PDF → See summary
  • Ask questions → Get AI responses
  • Click speaker icon → TTS in Indian English

🔥 Performance

Pagify uses health checks and UptimeRobot to reduce cold start time on Render.

const preWarm = async () => {
  try {
    const res = await fetch("https://aipdfreader-0f7m.onrender.com/health");
    console.log("Pre-warm:", res.ok);
  } catch (err) {
    console.error("Pre-warm error:", err.message);
  }
};
